用 Python 3.11 + React + MySQL 构建一个企业年会活动抽奖管理系统，Docker Compose 部署。

技术栈：
- 后端：Python 3.11，FastAPI + SQLAlchemy + PyMySQL
- 前端：React 18 + Vite + Ant Design，抽奖动画使用 CSS Animation
- 数据库：MySQL 8.0
- 部署：Docker Compose

核心功能：
1. 参与人员：批量导入员工名单（工号/姓名/部门），支持 CSV，可设置参与抽奖的范围（全员/指定部门）和权重（如：在职5年以上可多1次抽奖机会）
2. 奖项设置：录入奖项（等级：一等奖/二等奖/幸运奖等/奖品名称/数量/每次抽取人数），按等级从低到高顺序抽取
3. 抽奖执行：大屏滚动动画模式，显示参与人姓名快速滚动，点击"停止"随机抽取N名中奖者，中奖者姓名和部门大字展示
4. 中奖记录：每轮抽奖结果自动保存（奖项/中奖人员/抽奖时间），已中奖者默认不参与后续轮次（可配置是否允许重复中奖）
5. 签到功能：活动开始前扫码/输入工号签到，只有签到人员才能参与抽奖，实时显示签到人数和签到率
6. 结果导出：抽奖结束后导出完整中奖名单（工号/姓名/部门/奖项/奖品），格式为 HTML 可打印，也可导出 CSV

init.sql 预置300名员工、5个奖项和签到数据。
docker compose up -d 后访问 http://localhost:8080。
